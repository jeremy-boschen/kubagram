KubeViz - Kubernetes Visualization Tool

Visualize your Kubernetes clusters

KubeViz is a kubectl plugin that generates diagrams showing resources and their relationships in your Kubernetes clusters.

View Demo Visualization

Features

  • Generate SVG and Mermaid diagrams
  • Filter by namespace or label selectors
  • Smart detection of key components
  • Show relationships between resources
  • Interactive preview mode

Installation

Install KubeViz as a kubectl plugin using Krew:

kubectl krew install kubeviz

Or run it directly from source:

git clone https://github.com/username/kubectl-kubeviz
cd kubectl-kubeviz
./kubectl-kubeviz

Usage Examples


# Visualize entire cluster
kubectl kubeviz

# Visualize resources in a specific namespace
kubectl kubeviz -n kube-system

# Visualize resources with a specific label
kubectl kubeviz -l app=nginx

# Output Mermaid format instead of SVG
kubectl kubeviz -o mermaid

# Preview the diagram in a browser
kubectl kubeviz -o preview

# Save to a specific file
kubectl kubeviz -f my-cluster-diagram.svg
